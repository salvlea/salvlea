## 🚀 Projects

### 🔐 [CyberSecOnto — Cybersecurity Ontology](https://github.com/salvlea/Semantic_Web)
An **OWL ontology** modeling the relationships between systems, vulnerabilities, attacks, and mitigations in the cybersecurity domain.
- Built with **Protégé**, **SWRL rules**, and **SPARQL** queries
- Automated population and inference with **Python + owlready2**
- Classifies critical vulnerabilities (CVSS > 8.0) and high-risk systems automatically

### ☁️ [Smart ATS — Cloud-Native CV Screening System](https://github.com/salvlea/Sistemi_Cloud)
A fully **serverless, event-driven** AWS application that automates CV screening and candidate ranking for recruiters.
- **7 AWS services**: S3, SQS, Lambda, DynamoDB, Cognito, API Gateway, CloudWatch
- Intelligent ranking algorithm (skills 60%, experience 30%, education 10%)
- Complete **CI/CD pipeline** with GitHub Actions and AWS SAM
- Frontend built with **Flask**

### 🤖 [LLM Repository Analyzer](https://github.com/salvlea/isd_LLM)
A **Streamlit** web app that analyzes GitHub repositories using Large Language Models.
- Integrates **OpenAI API**, **Groq API**, and **Hugging Face Transformers** (BART, DistilGPT2)
- Uses **GitHub API** to extract and analyze source code
- Allows users to select NLP models and generate intelligent responses about any codebase
